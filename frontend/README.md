# FarmChainX – Full Stack Web Application

FarmChainX is a role-based web application designed to connect farmers, distributors, and customers using a secure and scalable system.

## Tech Stack
- Backend: Spring Boot (Java 21)
- Frontend: React
- Database: MySQL (Railway)
- Security: Spring Security, JWT
- Tools: STS, VS Code, GitHub

## Features
- User Login & Authentication
- Role-based dashboards (Farmer, Distributor, Admin)
- RESTful APIs
- Frontend–Backend integration

## Project Structure
/backend   → Spring Boot backend  
/frontend  → React frontend  

## How to Run
1. Start backend (port 8080)
2. Start frontend (port 3000)
3. Open browser and login
